User Signup From Password Field On for enable and off for disable password field

Enable to show password field within the signup from. Disable to have the password email to user after signup. it`s recommended to leave enabled.

Instant-Signup Username FieldOn for enable and off for disable username field

Enable to allow user to set their own username within the instant signup from. if disable a username is auto-genrated during signup. instant signup show during listing and review submission if the user isn`t signed up

Page Title(H1)On for enable and off for disable page title

This field dynamically sets the page title for each page to optimize search engine rankings and enhance user relevance.

Custom CSSPast your CSS code here.

Enter your Custom CSS code in the field. Do not include any tags or HTML in the field. Custom CSS entered here will override the theme CSS. In some cases, the fimportant tag may be needed. All CSS here will be kept during a theme update. We highly recommend adding your Custom CSS to a Child Theme.

Custom JSPaste your JS code here.

Enter your Custom JS Code in the field. Custom JS entered here will be kept during a theme update. We highly recommend entering Custom JS in a Child Theme.
